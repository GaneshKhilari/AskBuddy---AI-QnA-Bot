# 🧠 AskBuddy - AI QnA Bot

An AI-powered chatbot built using **LangChain**, **Google Gemini API**, and **Streamlit**.  
It allows users to ask questions and receive intelligent, real-time responses.

---

## 🚀 Live Demo
👉 https://askbuddy---ai-qna-bot.streamlit.app/

---

## ✨ Features
- 💬 Chat-based interface
- ⚡ Real-time AI responses
- 🧠 Session-based chat memory
- 🎯 Simple and user-friendly UI

---

## 🛠️ Tech Stack
- Python
- LangChain
- Google Gemini API
- Streamlit
